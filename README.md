## Instruções rápidas

1. Copie a pasta `PI_Grupo21_AgendaEstudos_final` para `C:\xampp\htdocs\`.
2. Inicie Apache e MySQL no XAMPP.
3. Acesse no navegador:

	`http://localhost/PI_Grupo21_AgendaEstudos_final/public/`

Caso o localhost não funcione deve alterar a porta em 'Config' Apache (httpd.conf) procure por 'Listen 80' e altere para 'Listen 8050'

	'http://127.0.0.1:8050/PI_Grupo21_AgendaEstudos_final/public/'

4. O sistema criará automaticamente o banco `db_estudos` e as tabelas (se não existirem).

5. Usuário de exemplo criado: **admin@example.com** / senha **senha123** (altere após login).
